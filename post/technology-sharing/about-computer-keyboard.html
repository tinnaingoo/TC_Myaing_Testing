ကွန်ပျူတာကီးဘုတ် (Computer Keyboard) အကြောင်း

By tinnaingoo • Feb 23, 2025

ကွန်ပျူတာကီးဘုတ်က ကွန်ပျူတာစနစ်တစ်ခုအတွက် အရေးပါသော Input Device တစ်ခုဖြစ်ပါတယ်။ ကီးဘုတ်မှတစ်ဆင့် စာရိုက်ခြင်း၊ ကွန်ပျူတာကို ထိန်းချုပ်ခြင်း၊ နှင့် အခြားလုပ်ဆောင်ချက်များကို လုပ်ဆောင်နိုင်ပါတယ်။ ကီးဘုတ်များသည် နည်းပညာတိုးတက်မှုနှင့်အတူ အမျိုးအစားများစွာ ပြောင်းလဲလာခဲ့ပြီး၊ ယနေ့ခေတ်တွင် လူသုံးများသော ပစ္စည်းတစ်ခုဖြစ်လာသည်။

ကီးဘုတ်၏ သမိုင်းကြောင်း

ကီးဘုတ်၏ သမိုင်းကြောင်းသည် ၁၉ ရာစုနှောင်းပိုင်းကာလများတွင် စတင်ခဲ့သည်။ ပထမဆုံးသော ကီးဘုတ်များကို Typewriter များတွင် အသုံးပြုခဲ့သည်။ ၁၈၆၈ ခုနှစ်တွင် Christopher Latham Sholes သည် QWERTY ကီးဘုတ်ကို တီထွင်ခဲ့ပြီး၊ ယနေ့ခေတ် ကီးဘုတ်များ၏ အခြေခံဖြစ်လာခဲ့သည်။

ကွန်ပျူတာကီးဘုတ်များသည် ၁၉၇၀ ခုနှစ်များတွင် စတင်ပေါ်ထွန်းလာခဲ့ပြီး၊ ပထမဆုံးသော ကွန်ပျူတာကီးဘုတ်များကို IBM နှင့် Apple တို့က တီထွင်ခဲ့သည်။ ယနေ့ခေတ်တွင် ကီးဘုတ်များသည် အမျိုးအစားများစွာ ရှိလာပြီး၊ လူသုံးများသော ပစ္စည်းတစ်ခုဖြစ်လာသည်။

ကီးဘုတ်၏ အမျိုးအစားများ

ကီးဘုတ်များကို အသုံးပြုမှုပုံစံနှင့် ဒီဇိုင်းပေါ်မူတည်၍ အမျိုးအစားများစွာ ခွဲခြားနိုင်သည်။

၁။ QWERTY ကီးဘုတ်

  • QWERTY ကီးဘုတ်သည် လူသုံးအများဆုံး ကီးဘုတ်အမျိုးအစားဖြစ်သည်။

  • ကီးဘုတ်၏ ထိပ်ဆုံးတွင် Q, W, E, R, T, Y ဟူသော စာလုံးများ ပါဝင်သည်။

  • Typewriter များတွင် အသုံးပြုခဲ့သော ဒီဇိုင်းကို အခြေခံထားသည်။

၂။ Ergonomic ကီးဘုတ်

  • လူသားများ၏ လက်နှင့် လက်ကောက်ဝတ်ကို သက်တောင့်သက်သာရှိစေရန် ဒီဇိုင်းထုတ်ထားသည်။

  • လက်ကောက်ဝတ်နာခြင်းကို ကာကွယ်ပေးသည်။

၃။ Gaming ကီးဘုတ်

  • ဂိမ်းကစားသူများအတွက် အထူးဒီဇိုင်းထုတ်ထားသည်။

  • RGB Backlighting နှင့် Macro Keys များ ပါဝင်သည်။

၄။ Wireless ကီးဘုတ်

  • Bluetooth သို့မဟုတ် USB Receiver များဖြင့် ချိတ်ဆက်အသုံးပြုနိုင်သည်။

  • ကြိုးမဲ့သုံးနိုင်သောကြောင့် သယ်ဆောင်ရလွယ်ကူသည်။

၅။ Virtual ကီးဘုတ်

  • စမတ်ဖုန်းများနှင့် တက်ဘလက်များတွင် အသုံးပြုသော ကီးဘုတ်အမျိုးအစားဖြစ်သည်။

  • ဖန်သားပြင်ပေါ်တွင် ပေါ်လာသော ကီးဘုတ်ဖြစ်သည်။

ကီးဘုတ်၏ အစိတ်အပိုင်းများ

ကီးဘုတ်တစ်ခုတွင် အဓိက အစိတ်အပိုင်းများစွာ ပါဝင်သည်။

၁။ Function Keys (F1-F12)

  • လုပ်ဆောင်ချက်များကို အမြန်လုပ်ဆောင်နိုင်ရန် အသုံးပြုသည်။

  • ဥပမာ - F1 သည် Help အတွက် အသုံးပြုသည်။

၂။ Alphanumeric Keys

  • စာလုံးများ (A-Z) နှင့် ဂဏန်းများ (0-9) ပါဝင်သည်။

၃။ Modifier Keys

  • Shift, Ctrl, Alt ကဲ့သို့သော ကီးများဖြစ်ပြီး၊ အခြားကီးများနှင့် တွဲဖက်အသုံးပြုသည်။

၄။ Navigation Keys

  • Arrow Keys, Home, End, Page Up, Page Down ကဲ့သို့သော ကီးများဖြစ်သည်။

၅။ Numeric Keypad

  • ဂဏန်းများကို အမြန်ရိုက်နိုင်ရန် ဒီဇိုင်းထုတ်ထားသည်။

ကီးဘုတ်၏ အလုပ်လုပ်ပုံ

ကီးဘုတ်သည် လျှပ်စစ်ဆိုင်ရာ နည်းပညာကို အသုံးပြု၍ အလုပ်လုပ်သည်။ ကီးတစ်ခုကို နှိပ်လိုက်သောအခါ၊ ကီးဘုတ်အတွင်းရှိ Circuit များကို လှုံ့ဆော်ပေးပြီး၊ ကွန်ပျူတာထံသို့ Signal များပေးပို့သည်။ ကွန်ပျူတာသည် ထို Signal များကို ဖတ်ရှုပြီး၊ သက်ဆိုင်ရာ စာလုံးသို့မဟုတ် လုပ်ဆောင်ချက်ကို ပြသပေးသည်။

ကွန်ပျူတာအသုံးပြုသူများအတွက် ကီးဘုတ်ပေါ်ရှိ Key များနှင့် Keyboard Shortcut များသည် အလုပ်လုပ်ဆောင်မှုကို မြန်ဆန်စေပြီး၊ အချိန်ကုန်သက်သာစေသည်။ ဤဆောင်းပါးတွင် အသုံးများသော Key များနှင့် Keyboard Shortcut များကို အသေးစိတ် ဖော်ပြထားပါသည်။

အသုံးများသော Key များ

၁။ Function Keys (F1-F12)

  • F1: Help ဖွင့်ရန်။

  • F2: File သို့မဟုတ် Folder အမည်ပြောင်းရန်။

  • F5: Page ကို Refresh လုပ်ရန်။

  • F11: Full Screen Mode သို့ ပြောင်းရန်။

၂။ Modifier Keys

  • Shift: စာလုံးကြီးရိုက်ရန် သို့မဟုတ် အထူးသင်္ကေတများရိုက်ရန်။

  • Ctrl (Control): Shortcut များနှင့် တွဲဖက်အသုံးပြုသည်။

  • Alt (Alternate): Menu များကို ရွေးချယ်ရန်။

  • Windows Key: Start Menu ဖွင့်ရန်။

၃။ Navigation Keys

  • Arrow Keys: Cursor ကို ရွှေ့ရန်။

  • Home: စာသားအစသို့ သွားရန်။

  • End: စာသားအဆုံးသို့ သွားရန်။

  • Page Up/Page Down: Page ကို အပေါ်/အောက်သို့ ရွှေ့ရန်။

၄။ Special Keys

  • Enter: Command သို့မဟုတ် စာပိုဒ်အသစ်စရိုက်ရန်။

  • Spacebar: Space ထည့်ရန်။

  • Backspace: စာလုံးများကို ဖျက်ရန်။

  • Delete: စာလုံးများကို ဖျက်ရန်။

  • Tab: Indent ထည့်ရန် သို့မဟုတ် Form များတွင် Field ပြောင်းရန်။

 ၁။ အထွေထွေ Shortcut များ

  • Ctrl + C: Copy လုပ်ရန်။

  • Ctrl + X: Cut လုပ်ရန်။

  • Ctrl + V: Paste လုပ်ရန်။

  • Ctrl + Z: Undo လုပ်ရန်။

  • Ctrl + Y: Redo လုပ်ရန်။

  • Ctrl + A: အားလုံးကို Select လုပ်ရန်။

  • Ctrl + S: File ကို Save လုပ်ရန်။

  • Ctrl + P: Print ထုတ်ရန်။

  • Ctrl + F: Find လုပ်ရန်။

  • Ctrl + N: New Window ဖွင့်ရန်။

  • Ctrl + W: Window ကို ပိတ်ရန်။

  • Alt + Tab: Open Program များကြား ပြောင်းရန်။

  • Alt + F4: Program ကို ပိတ်ရန်။

  • Windows + D: Desktop ကို ပြရန်။

  • Windows + L: Lock Screen လုပ်ရန်။

၂။ Browser Shortcut များ

  • Ctrl + T: New Tab ဖွင့်ရန်။

  • Ctrl + W: Tab ကို ပိတ်ရန်။

  • Ctrl + Shift + T: ပိတ်ထားသော Tab ကို ပြန်ဖွင့်ရန်။

  • Ctrl + Tab: Tab များကြား ပြောင်းရန်။

  • Ctrl + R: Page ကို Refresh လုပ်ရန်။

  • Ctrl + H: History ဖွင့်ရန်။

  • Ctrl + J: Download ဖိုင်များကို ကြည့်ရန်။

၃။ Text Editing Shortcut များ

  • Ctrl + B: Bold လုပ်ရန်။

  • Ctrl + I: Italic လုပ်ရန်။

  • Ctrl + U: Underline လုပ်ရန်။

  • Ctrl + Shift + >/<: Font Size ကြီးခြင်း/သေးခြင်း။

  • Ctrl + L: Text ကို Left Align လုပ်ရန်။

  • Ctrl + E: Text ကို Center Align လုပ်ရန်။

  • Ctrl + R: Text ကို Right Align လုပ်ရန်။

၄။ Windows Shortcut များ

  • Windows + E: File Explorer ဖွင့်ရန်။

  • Windows + R: Run Dialog Box ဖွင့်ရန်။

  • Windows + I: Settings ဖွင့်ရန်။

  • Windows + S: Search Bar ဖွင့်ရန်။

  • Windows + Shift + S: Screenshot ရိုက်ရန်။

  • Windows + . (သို့မဟုတ် ;): Emoji Panel ဖွင့်ရန်။

၅။ Mac Shortcut များ

  • Command + C: Copy လုပ်ရန်။

  • Command + V: Paste လုပ်ရန်။

  • Command + Z: Undo လုပ်ရန်။

  • Command + Shift + Z: Redo လုပ်ရန်။

  • Command + Q: Program ကို ပိတ်ရန်။

  • Command + Space: Spotlight Search ဖွင့်ရန်။

  • Command + Tab: Open Program များကြား ပြောင်းရန်။

Keyboard Shortcut များသည် ကွန်ပျူတာအသုံးပြုသူများအတွက် အချိန်ကုန်သက်သာစေပြီး၊ အလုပ်လုပ်ဆောင်မှုကို ပိုမိုမြန်ဆန်စေသည်။ ထို့ကြောင့်၊ ဤ Shortcut များကို မှတ်သားထားခြင်းဖြင့်၊ ကွန်ပျူတာအသုံးပြုမှုကို ပိုမိုထိရောက်စေနိုင်သည်။

Keyboard များနှင့် ချိတ်ဆက်ခဲ့သော Cable များ

ကွန်ပျူတာ Keyboard များသည် ကွန်ပျူတာစနစ်နှင့် ချိတ်ဆက်ရန် မတူညီသော Cable များကို အသုံးပြုခဲ့ကြသည်။ နည်းပညာတိုးတက်မှုနှင့်အတူ၊ ချိတ်ဆက်မှုပုံစံများလည်း ပြောင်းလဲလာခဲ့သည်။ ဤဆောင်းပါးတွင် Keyboard များနှင့် ကွန်ပျူတာကို ချိတ်ဆက်ခဲ့သော Cable များအကြောင်းကို အသေးစိတ် ဖော်ပြထားပါသည်။

၁။ PS/2 Cable

  • ပုံစံ: လုံးဝိုင်းပြီး၊ 6-pin connector ဖြစ်သည်။

  • အရောင်: Keyboard အတွက် ခရမ်းရောင်၊ Mouse အတွက် အစိမ်းရောင်။

  • အသုံးပြုမှု: 1990 ခုနှစ်များတွင် အသုံးများခဲ့ပြီး၊ ယခုအခါ အသုံးနည်းလာသည်။

  • အားသာချက်: Hardware Interrupt ကို အသုံးပြုထားသောကြောင့်၊ Input များကို ပိုမိုတိကျစွာ လက်ခံနိုင်သည်။

  • အားနည်းချက်: Hot-Swapping မပံ့ပိုးပါ၊ USB ထက် နှေးကွေးသည်။

၂။ USB Cable

  • ပုံစံ: လေးထောင့်ပုံစံ၊ Type-A, Type-B, Type-C စသည်ဖြင့် အမျိုးမျိုးရှိသည်။

  • အသုံးပြုမှု: ယနေ့ခေတ်တွင် အသုံးအများဆုံး ချိတ်ဆက်မှုပုံစံဖြစ်သည်။

  • အားသာချက်: Plug-and-Play စနစ်ကို အသုံးပြုထားသောကြောင့်၊ လွယ်ကူစွာ ချိတ်ဆက်နိုင်သည်။

  • အမြန်နှုန်း: USB 2.0 (480 Mbps), USB 3.0 (5 Gbps), USB 3.1 (10 Gbps), USB4 (40 Gbps)။

၃။ Serial Port Cable

  • ပုံစံ: လေးထောင့်ပုံစံ၊ 9-pin သို့မဟုတ် 25-pin connector ဖြစ်သည်။

  • အသုံးပြုမှု: 1980 ခုနှစ်များတွင် အသုံးများခဲ့ပြီး၊ ယခုအခါ အသုံးနည်းလာသည်။

  • အားသာချက်: ရိုးရှင်းပြီး စိတ်ချရသော ချိတ်ဆက်မှုကို ပေးစွမ်းနိုင်သည်။

  • အားနည်းချက်: အမြန်နှုန်းနှေးကွေးပြီး၊ ခေတ်မီ Device များနှင့် မကိုက်ညီပါ။

၄။ ADB (Apple Desktop Bus) Cable

  • ပုံစံ: လေးထောင့်ပုံစံ၊ Apple ကွန်ပျူတာများအတွက် အထူးဒီဇိုင်းထုတ်ထားသည်။

  • အသုံးပြုမှု: 1980 နှင့် 1990 ခုနှစ်များတွင် Apple ကွန်ပျူတာများတွင် အသုံးပြုခဲ့သည်။

  • အားသာချက်: Keyboard, Mouse, နှင့် အခြား Device များကို တစ်ချိန်တည်းတွင် ချိတ်ဆက်နိုင်သည်။

  • အားနည်းချက်: Apple ကွန်ပျူတာများအတွက်သာ သီးသန့်ဒီဇိုင်းထုတ်ထားသည်။

၅။ Wireless Connection (Bluetooth/USB Receiver)

  • ပုံစံ: Cable မလိုဘဲ၊ Wireless ချိတ်ဆက်မှုကို အသုံးပြုသည်။

  • အသုံးပြုမှု: ယနေ့ခေတ်တွင် အသုံးများလာသော ချိတ်ဆက်မှုပုံစံဖြစ်သည်။

  • အားသာချက်: Cable မလိုဘဲ၊ လွတ်လပ်စွာ အသုံးပြုနိုင်သည်။

  • အားနည်းချက်: Battery လိုအပ်ပြီး၊ အချို့သော Wireless Keyboard များသည် Input Lag ရှိနိုင်သည်။

ယနေ့ခေတ်တွင် USB နှင့် Wireless (Bluetooth/USB Receiver) ချိတ်ဆက်မှုပုံစံများသည် Keyboard များအတွက် အသုံးအများဆုံးဖြစ်လာသည်။ USB Type-C သည် မြန်နှုန်းမြင့်မားပြီး Reversible ဖြစ်သောကြောင့်၊ ခေတ်ပေါ် Keyboard များတွင် ပိုမိုအသုံးများလာပါတယ်။

အခုဆိုရင် Computer Keyboard နဲ့ ပက်သက်ပြီး ဗဟုသုတအချို့ကို ရရှိကြလိမ့်မယ်လို့ မျှေလင့်မိပါတယ်။
နောက်ထပ်လည်း အခုလို ဗဟုသုဖြစ်စရ အကြောင်းအရာလေးတွေကို တင်ပေးဦးမှာမို့ လာရောက်ပေးကြဖို့ မေတ္တာရပ်ခံပါတယ်။
