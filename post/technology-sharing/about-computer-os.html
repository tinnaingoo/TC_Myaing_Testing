Operating System (OS) အကြောင်း

By tinnaingoo • Feb 23, 2025

Operating System (OS) ဆိုသည်မှာ user နှင့် computer စက်ပစ္စည်းများအကြား ဆက်သွယ်မှု အဖြစ် လုပ်ဆောင်သော software တစ်ခုဖြစ်သည်။ အခြား application များ (Chrome, MS Word, Games စသည်) ကို run လုပ်ရန်နှင့် လုပ်ဆောင်ချက်များကို ဆောင်ရွက်ရန် OS က environment ပေးထားပြီး၊ computer ၏ hardware နှင့် software တို့ကို ချိတ်ဆက်စီမံခန့်ခွဲပေးသည်။ အဲဒါကြောင့် computer သို့မဟုတ် mobile device တစ်ခုတွင် OS မရှိဘဲ အသုံးပြု၍မရနိုင်ပါ။

1. History Of OS

၁၉၅၀ နှောင်းပိုင်းတွင် tape storage ကို စီမံရန် OS များကို စတင်တီထွင်ခဲ့ပြီး၊ General Motors Research Lab သည် ၁၉၅၀ ပြည့်နှစ်များအစောပိုင်းတွင် IBM 701 အတွက် ပထမဆုံး OS ကို အကောင်အထည်ဖော်ခဲ့သည်။ ၁၉၆၀ ပြည့်လွန်နှစ်များအလယ်ပိုင်းတွင် disk storage ကို အသုံးပြုစတင်ပြီး၊ ၁၉၆၀ ပြည့်လွန်နှစ်များနှောင်းပိုင်းတွင် Unix OS ၏ ပထမဆုံးမူကွဲကို တီထွင်ခဲ့ကြသည်။ Microsoft ၏ ပထမဆုံး OS သည် DOS ဖြစ်ပြီး၊ Seattle ကုမ္ပဏီမှ 86-DOS software ကို ဝယ်ယူပြီး ၁၉၈၁ ခုနှစ်တွင် တည်ဆောက်ခဲ့သည်။ နောက်ပြီး Windows OS ကို ၁၉၈၅ ခုနှစ်တွင် GUI (Graphical User Interface) အခြေခံ၍ MS-DOS နှင့် တွဲဖက်အသုံးပြုခဲ့သည်။

2. Operating System ၏ Features

OS များသည် အောက်ပါအဓိက လုပ်ဆောင်ချက်များကို ဆောင်ရွက်ပေးသည်။

  • Protected and Supervisor Mode: Hardware resources များကို ထိန်းချုပ်ရန်
  • Disk Access and File Systems: File management နှင့် device drivers
  • Networking and Security: Network connections နှင့် data protection
  • Program Execution: Process management, scheduling, and context switching
  • Memory Management: Virtual memory, allocation, and multitasking
  • I/O Operations: Input/Output device handling
  • Error Detection and Resource Allocation: System stabilityနှင့် performance

3. What is a Kernel?

Kernel သည် operating system ၏ အတွင်းအကျဆုံးအစိတ်အပိုင်းဖြစ်ပြီး၊ hardware နှင့် software အကြား ဆက်သွယ်မှုကို စီမံခန့်ခွဲရန် တာဝန်ရှိသည်။ OS ၏ shell သည် user interface အပြင်ဘက်တွင် ရှိသော်လည်း kernel သည် core functionality များကို low-level scheduling, inter-process communication, process synchronization နှင့် context switching စသည့် လုပ်ငန်းများကို လုပ်ဆောင်သည်။

Kernel အမျိုးအစားများ

  • Monolithic Kernel: အားလုံးသော OS services များကို single code block အဖြစ် ဖော်ပြပြီး၊ hardware နှင့် software အကြား distinct communication layer ရှိစေသည်။
  • Microkernel: Kernel services များကို အနည်းငယ်သာ kernel space တွင်ထား၍၊ user services များကို different address space တွင် run နိုင်ရန် ခွဲထုတ်ထားသည်။

4. Operating System ၏ လုပ်ဆောင်ချက်များ

  • Process Management

        Process များကို create, delete, synchronize နှင့် communicate လုပ်ရန်

  • Memory Management

        Program များအတွက် memory allocation နှင့် de-allocation

  • File Management

        Files များကို organize, store, retrieve, name, share နှင့် protect လုပ်ခြင်း

  • Device Management

        Input/Output devices များကို track, allocate နှင့် de-allocate လုပ်ခြင်း

  • I/O System Management

        Input/Output devices အတွက် driver နှင့် interface များဖြင့် စီမံခြင်း

  • Secondary Storage Management

        Primary, secondary နှင့် cache storage များ၏ သုံးစွဲမှုကို စီမံခြင်း

  • Security

        Malware နှင့် unauthorized access များမှ data ကို ကာကွယ်ခြင်း

  • Command Interpretation

        User commands များကို translate ပြုလုပ်၍ system resources ကို ခိုင်းခြင်း

  • Networking

        Distributed systems နှင့် network communication ကို စီမံခြင်း

  • Job Accounting

        Resource နှင့် time usage ကို track လုပ်ခြင်း

  • Communication Management

        အခြား computer များနှင့် Devices များအကြား ဆက်သွယ်မှု

5. Firmware နှင့် Operating System အကြား ကွာခြားချက်

     Smartphone တွေမှာရှိတဲ့ Android တို့ IOS တို့ Window တို့တွေဟာအမှန်တကယ်တော့ OS တွေပါ။ Firmware မဟုတ်ပါဘူး။ OS နဲ့ Firmware တစ်ခါတည်းတွဲပါတာတွေ အချို့ Phone တွေတော့ရှိတယ်။

7. OS အသုံးပြုမှုနှင့် Computer / Mobile အမျိုးအစားအလိုက် ရေပန်းစားမှု

Computer များ: Windows, macOS နှင့် Linux distributions များ သုံးဆောင်ကြသည်။ Desktop နှင့် Laptop များတွင် အမျိုးအစားအလိုက် OS များကို optimize ပြုလုပ်ထားသည်။

Smartphones: Android နှင့် iOS ကို အဓိက အသုံးပြုကြပြီး၊ touch-based interface နှင့် low power consumption အတွက် အထူးသင့်လျော်သည်။

8. OS အသုံးပြုနိုင်ရန် System လိုအပ်ချက်များ

OS တစ်ခုချင်းစီသည် hardware (CPU, RAM, Storage, GPU) အတွက် သတ်မှတ်ထားသော requirements များရှိသည်။ ဥပမာ Windows 11 သည် 64-bit CPU, 4GB RAM, 64GB storage, TPM 2.0 နှင့် Secure Boot support များလိုအပ်သည်။ macOS သည် Apple hardware တွင်သာ run ဖြစ်သည်။

9. ယနေ့ခေတ် OS အသုံးပြုမှုနှင့် အကြံပြုချက်

ယနေ့ခေတ်တွင် Desktop နှင့် Laptop များအတွက် Windows 10/11, macOS, Linux distributions များ၊ Server များအတွက် Linux-based OS များနှင့် Windows Server, Mobile devices အတွက် Android နှင့် iOS ကို အထူးသင့်လျော်စွာ အသုံးပြုကြသည်။
