Computer ဆိုတာဘာလဲ

By tinnaingoo • Feb 23, 2025

ကွန်ပျူတာဆိုသည်မှာ အချက်အလက်များကို လျင်မြန်စွာနှင့် တိကျစွာ လုပ်ဆောင်နိုင်သော လျှပ်စစ်စက်ကိရိယာတစ်မျိုးဖြစ်သည်။ ကွန်ပျူတာသည် လူသားတို့၏ လုပ်ဆောင်ချက်များကို အထောက်အကူပြုရန် ဒီဇိုင်းထုတ်ထားသော စက်ဖြစ်ပြီး၊ သတင်းအချက်အလက်များကို သိမ်းဆည်းခြင်း၊ ပြန်လည်ရယူခြင်း၊ နှင့် လုပ်ဆောင်ခြင်းတို့ကို ပြုလုပ်နိုင်သည်။ ကွန်ပျူတာသည် လူသားတို့၏ ဘဝကို အများကြီး အထောက်အကူပြုပြီး၊ နည်းပညာတိုးတက်မှုနှင့်အတူ လူ့အဖွဲ့အစည်းတစ်ခုလုံးကို ပြောင်းလဲစေခဲ့သည်။

ကွန်ပျူတာတွေက လူသားတွေကို ဘယ်လိုတွေအကျိုးပြုနိုင်သလဲ

ကွန်ပျူတာများသည် လူသားတို့အတွက် အကျိုးကျေးဇူးများစွာကို ဖြစ်စေသည်။ ပထမဦးစွာ၊ ကွန်ပျူတာများသည် အချက်အလက်များကို လျင်မြန်စွာနှင့် တိကျစွာ လုပ်ဆောင်နိုင်သောကြောင့်၊ လုပ်ငန်းဆောင်တာများကို ပိုမိုမြန်ဆန်စွာ ပြီးမြောက်စေသည်။ ဒုတိယအနေဖြင့်၊ ကွန်ပျူတာများသည် သတင်းအချက်အလက်များကို သိမ်းဆည်းခြင်းနှင့် ပြန်လည်ရယူခြင်းတို့ကို လွယ်ကူစေသည်။ တတိယအနေဖြင့်၊ ကွန်ပျူတာများသည် ဆက်သွယ်ရေးနှင့် သတင်းအချက်အလက်များကို မျှဝေခြင်းတို့ကို လွယ်ကူစေသည်။ နောက်ဆုံးအနေဖြင့်၊ ကွန်ပျူတာများသည် ပညာရေးနှင့် သုတေသနလုပ်ငန်းများကို အထောက်အကူပြုသည်။

ကွန်ပျူတာကို ဘယ်သူက ဘယ်နေရာမှာ ဘယ်နှစ်လောက်ကစတင်ခဲ့တာလဲ

ကွန်ပျူတာကို စတင်တီထွင်ခဲ့သူမှာ Charles Babbage ဖြစ်ပြီး၊ သူသည် 1822 ခုနှစ်တွင် Analytical Engine ကို တီထွင်ခဲ့သည်။ သို့သော်၊ ပထမဆုံးသော အီလက်ထရွန်းနစ် ကွန်ပျူတာကို 1940 ခုနှစ်များတွင် တီထွင်ခဲ့သည်။ ENIAC (Electronic Numerical Integrator and Computer) သည် ပထမဆုံးသော အီလက်ထရွန်းနစ် ကွန်ပျူတာဖြစ်ပြီး၊ 1945 ခုနှစ်တွင် တီထွင်ခဲ့သည်။

ကွန်ပျူတာတွေ စတင်တီထွင်ဖို့ ဘယ်လိုတွေစိတ်ကူးရခဲ့တာလဲ

ကွန်ပျူတာတွေ စတင်တီထွင်ဖို့ စိတ်ကူးရခဲ့သည်မှာ လူသားတို့၏ လုပ်ဆောင်ချက်များကို အထောက်အကူပြုရန် ဖြစ်သည်။ ပထမဦးစွာ၊ သင်္ချာနှင့် သိပ္ပံပညာရပ်များတွင် ရှုပ်ထွေးသော တွက်ချက်မှုများကို လျင်မြန်စွာ လုပ်ဆောင်နိုင်ရန် လိုအပ်ချက်ကြောင့် ကွန်ပျူတာကို တီထွင်ခဲ့သည်။ ဒုတိယအနေဖြင့်၊ စီးပွားရေးနှင့် စာရင်းအင်းများကို စနစ်တကျ စီမံခန့်ခွဲနိုင်ရန် လိုအပ်ချက်ကြောင့် ကွန်ပျူတာကို တီထွင်ခဲ့သည်။

ကွန်ပျူတာဆင့်ကဲပြောင်းလဲလာပုံနဲ့ တစ်ဆင့်ခြင်းစီရဲ့ အလုပ်လုပ်ပုံကွာခြားချက်၊ ပုံစံနဲ့အသုံးဝင်မှု

ကွန်ပျူတာများသည် ဆင့်ကဲပြောင်းလဲလာခဲ့ပြီး၊ တစ်ဆင့်ခြင်းစီတွင် အလုပ်လုပ်ပုံ၊ ပုံစံနှင့် အသုံးဝင်မှုများ ကွာခြားခဲ့သည်။

  1. ပထမမျိုးဆက် (1940-1956): Vacuum Tubes များကို အသုံးပြုထားသော ကွန်ပျူတာများ။ ဥပမာ - ENIAC။

  2. ဒုတိယမျိုးဆက် (1956-1963): Transistors များကို အသုံးပြုထားသော ကွန်ပျူတာများ။

  3. တတိယမျိုးဆက် (1964-1971): Integrated Circuits (ICs) များကို အသုံးပြုထားသော ကွန်ပျူတာများ။

  4. စတုတ္ထမျိုးဆက် (1971-ယနေ့): Microprocessors များကို အသုံးပြုထားသော ကွန်ပျူတာများ။

မြန်မာနိုင်ငံမှာ ကွန်ပျူတာကို ဘယ်အချိန်မှ စတင်အသုံးပြုပြီး အများအပြားဘယ်အချိန်က အသုံးပြုခဲ့သလဲ

မြန်မာနိုင်ငံတွင် ကွန်ပျူတာကို 1970 ခုနှစ်များတွင် စတင်အသုံးပြုခဲ့သည်။ သို့သော်၊ 1990 ခုနှစ်များတွင် ကွန်ပျူတာများကို အများအပြား အသုံးပြုလာခဲ့သည်။ ယနေ့ခေတ်တွင်၊ ကွန်ပျူတာများသည် မြန်မာနိုင်ငံ၏ ပညာရေး၊ စီးပွားရေးနှင့် လူမှုရေးကဏ္ဍများတွင် အရေးပါသော အခန်းကဏ္ဍမှ ပါဝင်နေသည်။

ယနေ့ခေတ်ကွန်ပျူတာတွေနဲ့ ကွန်ပျူတာအမျိုးအစား၊ Brand များ

ယနေ့ခေတ်တွင်၊ ကွန်ပျူတာများသည် အမျိုးအစားများစွာရှိပြီး၊ လူသုံးများသော Brand များမှာ Dell, HP, Lenovo, Apple, Asus, Acer တို့ဖြစ်သည်။

ကွန်ပျူတာတွေက ဘယ်လိုအလုပ်လုပ်သလဲ

ကွန်ပျူတာများသည် Input, Processing, Output, နှင့် Storage ဟူသော အဆင့်လေးဆင့်ဖြင့် အလုပ်လုပ်သည်။

  1. Input: ကွန်ပျူတာထဲသို့ အချက်အလက်များကို ထည့်သွင်းခြင်း။

  2. Processing: အချက်အလက်များကို လုပ်ဆောင်ခြင်း။

  3. Output: လုပ်ဆောင်ပြီးသော အချက်အလက်များကို ထုတ်ပေးခြင်း။

  4. Storage: အချက်အလက်များကို သိမ်းဆည်းခြင်း။

ကွန်ပျူတာအတွင်းမှာ ပါဝင်တဲ့ပစ္စည်းများနဲ့ အလုပ်လုပ်ပုံ

ကွန်ပျူတာအတွင်းတွင် ပါဝင်သော ပစ္စည်းများမှာ -

  1. CPU (Central Processing Unit): ကွန်ပျူတာ၏ ဦးနှောက်ဖြစ်ပြီး၊ အချက်အလက်များကို လုပ်ဆောင်သည်။

  2. RAM (Random Access Memory): အချက်အလက်များကို ယာယီသိမ်းဆည်းထားသည်။

  3. Hard Drive: အချက်အလက်များကို အမြဲတမ်းသိမ်းဆည်းထားသည်။

  4. Motherboard: ကွန်ပျူတာ၏ အဓိက ပါဝင်ပစ္စည်းများကို ချိတ်ဆက်ထားသည်။

ကွန်ပျူတာတစ်ခုရဲ့ Basic Function များ

ကွန်ပျူတာတစ်ခု၏ Basic Function များမှာ -

  1. Data Input: အချက်အလက်များကို ထည့်သွင်းခြင်း။

  2. Data Processing: အချက်အလက်များကို လုပ်ဆောင်ခြင်း။

  3. Data Output: လုပ်ဆောင်ပြီးသော အချက်အလက်များကို ထုတ်ပေးခြင်း။

  4. Data Storage: အချက်အလက်များကို သိမ်းဆည်းခြင်း။

ကွန်ပျူတာအသုံးပြုသင့်ကြောင်း အကြံပြုချက်

ကွန်ပျူတာအသုံးပြုခြင်းသည် လူသားတို့၏ ဘဝကို အများကြီး အထောက်အကူပြုသည်။ ထို့ကြောင့်၊ ကွန်ပျူတာအသုံးပြုသင့်ပါတယ်။

  1. ပညာရေး: ကွန်ပျူတာသည် အလုပ်လုပ်ဆောင်မှု အကျိုးကျေးဇူး၊ ပညာရေး လေ့လာမှုနှင့် အချက်အလက် ထိန်းသိမ်းမှုတို့တွင် အရေးကြီးသော ကိရိယာဖြစ်သလို အချိန်နှင့် အရင်းအမြစ်များကို ထိရောက်စွာ စီမံခန့်ခွဲနိုင်စေသည်။

  2. စီးပွားရေး: Software Development, Data Analysis, Digital Marketing စသည့် နယ်ပယ်များတွင် ကွန်ပျူတာကို အသုံးပြုခြင်းဖြင့် စီးပွားရေးတိုးတက်စေပြီး ဖန်တီးမှုကို တိုးတက်စေသည်။

  3. ဆက်သွယ်ရေး: ကွန်ပျူတာများသည် ဆက်သွယ်ရေးကို လွယ်ကူစေသည်။

  4. သုတေသန: ကွန်ပျူတာများသည် သုတေသနလုပ်ငန်းများအတွက် များစွာအထောက်အကူပြုသည်။
