32 bit နဲ့ 64 bit က ဘာတွေကွာတာလဲ?

By tinnaingoo • Feb 28, 2025

ကွန်ပျူတာ အသုံးပြုသူတွေ အနေနဲ့ Windows တင်တဲ့အခါ (သို့) ဆော့ဖ်ဝဲလ်တွဲ ဂိမ်းတွေကို download ဆွဲတဲ့အခါမှာ 32 bit Version နှင့် 64 bit Version ကွဲနေတာ‌ကြောင့် စိတ်ရှုပ်ဖူးကြမယ် ထင်ပါတယ်။
Windows, Macintosh, Linux စတဲ့ Computer OS (operating system) တွေမှာ 32 bit နဲ့ 64 bit ဆိုပြီး အမျိုးအစား နှစ်မျိုး ရှိပါတယ်။
တစ်ကယ်တော့ 32 bit တို့ 64 bit တို့ဆိုတာ computer ရဲ့ CPU (central processing unit) က အချက်အလက်တွေကို ဘယ်လို ကိုင်တွယ်သလဲဆိုတဲ့ နည်းလမ်းတွေပဲ ဖြစ်ပါတယ်။
64 bit OS တွေက RAM (Random Access Memory) များများကို ကိုင်တွယ် ဖြေရှင်းနိုင်ပြီးတော့ 32 bit ထက်ပို စွမ်းဆောင်ရည် မြင့်ပါတယ်။

ဒါဆိုရင် ရှိသမျှ computer တွေအားလုံး 64 bit တင်လိုက်ရတော့မလား? အဲလိုလည်း မဟုတ်ပါဘူး။ စွမ်းဆောင်ရည် ပိုမြင်တဲ့ 64 bit ကို တင်မယ်ဆိုရင် 64 bit ကို မောင်းနှင်ပေးနိုင်တဲ့ CPU နဲ့မှ ဖြစ်ပါမယ်။
မရှားပေမယ့် စက်တိုင်းမှာ 64 bit run လို့မရပါဘူး။ ဒါကြောင့် 64 bit မမောင်းနှင်နိုင်တဲ့ computer မှာ 64 bit မတင်သင့်ပါဘူး။

ဒါဆို 64 bit မောင်းနှင်နိုင်တဲ့စက် ဟုတ်/မဟုတ် ဘယ်လိုစစ်မလဲ?

၁။ ဘာနည်းပညာမှ မလိုဘဲ စစ်ဆေးနည်းကတော့ ဆိုင်တွေမှာ စဝယ်ဝယ်ချင်း တင်ထားတဲ့ OS version ကိုအသုံးပြုတာ အကောင်းဆုံးပါပဲ။

ဝယ်တုန်းက 32 bit တင်ထားတယ်ဆိုရင် ပြန်တင်တဲ့အခါမှာလည်း 32 bit တင်ပါ။ ဝယ်တုန်းက 64 bit တင်ထားရင် ပြန်တင်တဲ့အခါ 64 bit ပဲတင်ပါ။

၂။ မဟုတ်သေးပါဘူး၊ နည်းပညာကျကျ သိချင်တာဆိုရင်တော့ အလွယ်နည်း တစ်ခုကတော့ ကိုယ့်ရဲ့ Windows မှာ (Mac, Linux မရပါ) SecurAble ဆိုတဲ့ software လေးတစ်ခုတင်ပြီး စစ်ကြည့်လို့ရပါတယ်။

SecurAble Software Download ဆွဲရန်

Download လုပ်ရတာလည်း file size မကြီးပါဘူး။ Download လုပ်ပြီး run လိုက်တဲ့အခါ



ဒီလိုပေါ်ရင် 32 bit အထိပဲ တင်လို့ရပါမယ်။


ဒီလိုပေါ်ရင်တော့ 64 bit windows တင်လို့ရပါတယ်။

၃။ ဒီ tool download လုပ်ဖို့ အဆင်မပြေဘူးဆိုရင်တော့ Keyboard ကနေ Windows Key + R လို့ ရိုက်ထည့်လိုက်ပါ။ ပြီးရင် Run Box ကျလာပါလိမ့်မယ်။



Run Box အတွင်းမှာ "msinfo32" လို့ရိုက်ထည့်ပြီး Ok ကို နှိပ်ပါ။


အခုလိုပေါ်လာတဲ့အခါ System Type မှာစစ်ဆေးကြည့်ပါ။
x64-based PC ဆိုရင် 64 bit တင်လို့ရပါတယ်။
x86-based PC ဆိုရင် 64 bit တင်လို့ရပါတယ်။
ဒီလောက်ဆိုရင် 32 bit နဲ့ 64 bit ကွာခြားချက်လေးတွေ အကြမ်းဖျင်းတော့ သိရှိသွားပြီး 32 bit လား 64 bit လားဆိုတာကို ခွဲခြားတက်ကြမယ်လို့ မျှော်လင့်မိပါတယ်။
